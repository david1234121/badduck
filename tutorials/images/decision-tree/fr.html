Vue d'ensemble

Cet arbre décisionnel décrit comment utiliser l’attribut alt de l’élément <img> dans diverses situations. Pour certains types d’images, des approches alternatives existent, comme utiliser des images CSS d’arrière-plan pour les images décoratives, ou des polices web plutôt que des images de texte.

  • Est-ce que l’image contient du texte ?
    • Oui :
      • … et le texte est aussi présent à proximité, comme du texte réel. Utilisez un attribut alt vide. Voir Images décoratives (en anglais).
      • … et le texte est seulement présenté pour des effets visuels. Utilisez un attribut alt vide. Voir Images décoratives (en anglais).
      • … et le texte a une fonction spécifique : il s’agit d’une icône par exemple. Utilisez l’attribut alt pour communiquer la fonction de l’image. Voir Images fonctionnelles (en anglais).
      • … et le texte dans l’image n’est pas présent ailleurs. Utilisez l’attribut alt pour inclure le texte de l’image. Voir Images de texte (en anglais).
    • Non :
      • Continuez.
  • Est-ce que l’image est utilisée dans un lien ou un bouton, et serait-il difficile voire impossible de comprendre ce que le lien ou le bouton fait, si l’image n’était pas présente ?
    • Oui:
      • Utilisez l’attribut alt pour communiquer la destination du lien ou de l’action réalisée. Voir Images fonctionnelles (en anglais).
    • Non :
      • Continuez.
  • Est-ce que l’image contribue à comprendre la signification de la page courante ou du contexte ?
    • Oui :
      • … et c’est une simple illustration ou photographie. Utilisez une description brève de l’image, d’une manière qui transmette cette signification dans l’attribut alt. Voir Images informatives (en anglais).
      • … et c’est un graphique ou une information complexe. Incluez l’information contenue dans l’image ailleurs sur la page. Voir Images complexes (en anglais).
      • … et elle présente du contenu qui est redondant avec le texte réel à proximité. Utilisez un attribut alt vide. Voir Images fonctionnelles (en anglais) (redondantes).
    • Non :
      • Continuez.
  • Est-ce que l’image est purement décorative, ou pas destinée aux utilisateurs ?
    • Oui :
      • Utilisez un attribut alt vide. Voir Images décoratives (en anglais).
    • Non :
      • Continuez.
  • L’usage de l’image n’est pas listé ci-dessus ; ou le texte alt à fournir n’est pas clair ?
    • Cet arbre décisionnel ne couvre pas tous les cas. Pour des informations détaillées sur les textes alternatifs à fournir, référez-vous à la page Concepts - Images (en anglais).
